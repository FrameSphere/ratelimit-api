# SEO Assets Todo

Die folgenden Bilder sollten noch erstellt und hinzugefügt werden für optimales SEO:

## Benötigte Images:

1. **og-image.png** (1200x630px)
   - Open Graph Image für Social Media Previews
   - Sollte Logo + Titel "RateLimit API" enthalten
   - Speicherort: `/public/og-image.png`

2. **favicon.png** (32x32px)
   - PNG Fallback für ältere Browser
   - Speicherort: `/public/favicon.png`

3. **apple-touch-icon.png** (180x180px)
   - Icon für iOS Home Screen
   - Speicherort: `/public/apple-touch-icon.png`

4. **favicon-192.png** (192x192px)
   - PWA Icon (klein)
   - Speicherort: `/public/favicon-192.png`

5. **favicon-512.png** (512x512px)
   - PWA Icon (groß)
   - Speicherort: `/public/favicon-512.png`

6. **screenshot.png** (Optional)
   - Screenshot der App für App Stores
   - Speicherort: `/public/screenshot.png`

## Schnell erstellen:

Du kannst Tools wie:
- Canva (für og-image.png)
- Favicon.io (für alle favicon Varianten)
- Oder das favicon.svg mit einem Converter in PNGs umwandeln

Das favicon.svg ist bereits vorhanden und kann als Basis verwendet werden!
